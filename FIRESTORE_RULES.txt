Actualizar las reglas de Firestore a:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /posts/{document=**} {
      allow read, write: if request.auth != null;
    }
    match /users/{userId} {
      allow read: if request.auth != null;
      allow create, delete: if request.auth != null && request.auth.uid == userId;
      allow update: if request.auth != null && (
        request.auth.uid == userId ||
        request.resource.data.diff(resource.data).affectedKeys().hasOnly(['following', 'followers'])
      );
    }
  }
}

IMPORTANTE: Estas reglas permiten:
1. Cualquier usuario autenticado puede leer cualquier perfil de usuario
2. Solo un usuario puede crear/eliminar su propio documento
3. Un usuario puede actualizar su propio documento completamente
4. Cualquier usuario autenticado puede actualizar los campos following/followers de cualquier usuario (para la funcionalidad de seguimiento)
